BEERCRAFT
HTML CSS Javascript Laravel Bootstrap 5 MySQL Responsive

About this project:

Commercial website of a beer producer called "Beercraft". It lists the different local breweries around the world and the different type of beers you can find there. A brewery owner can add his place on the website and can choose the beers he's selling from the whole list offered.

  • Authentication (Fortify)
  • Upload brewery
  • Relations: 1-1/1-N/N-N
  • CRUD: create/read/update/delete
  • Use of Factory/Seeders
  • Contact Form
  • Use of a Template
  • MVC: Model-View-Controller
  • Areas: Public/Private

Using a Template from Bootstrapmade prevents me to deploy the project, due to the Content-Security-Policy.

REACT MEETUPS APP
React React Hooks CSS HTML Javascript Firebase Responsive

About this project:

This is a Single-Page-Application made with React, where you can save your past and present Meetups (exhibitions, movies, parties, online events...) you have attended or plan to attend. You can save new Meetups, consult all your Meetups, save a Meetup as Favorite and update the Meetup if necessary.

Database: Firestore of Firebase to store the Meetups in a NoSQL document-oriented database

  • React-DOM
  • React-router
  • Components
  • UseEffect Hook
  • UseContext hook
  • UseState hook
  • useHistory hook
  • React build-in "Card" and "Link" Components
